üìñ Installation Instructions

üì± ATAK Installation

  1. Download the profile package (.zip) to your Android device
  2. Open ATAK
  3. Go to Settings ‚Üí Network Preferences ‚Üí Manage Server Connections
  4. Tap Import
  5. Select the downloaded .zip file
  6. Enter the certificate password when prompted
  7. The connection should activate automatically

üíª WinTAK Installation

  1. Download and extract the profile package
  2. Open WinTAK
  3. Go to Settings ‚Üí Network Preferences
  4. Click Import Connection
  5. Select the .pref file from the extracted folder
  6. When prompted, browse to and select the .p12 certificate file
  7. Enter the certificate password

üçé iTAK Installation

  1. Download the profile package
  2. Extract and AirDrop or email the .p12 file to your iOS device
  3. When the certificate profile appears, tap Install
  4. Enter the certificate password
  5. Open iTAK
  6. Go to Settings ‚Üí Servers
  7. Add a new server with the connection details shown above
